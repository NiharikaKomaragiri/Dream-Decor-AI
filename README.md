# 🏡 Dream Decor AI

Dream Decor AI is an AI-powered interior design application built with Streamlit and Hugging Face.

## Features

- 🎨 Choose interior design style
- 💰 Select budget
- 🌈 Choose theme color
- 🤖 Generate AI-designed room concepts
- 📥 Download generated images

## Technologies Used

- Python
- Streamlit
- Hugging Face
- Pillow
- python-dotenv

## Installation

```bash
git clone https://github.com/yourusername/Dream-Decor-AI.git

cd Dream-Decor-AI

pip install -r requirements.txt

streamlit run app.py
```
